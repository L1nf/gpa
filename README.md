# 正方教务爬虫

## 已实现功能：

* [X]  获取指定学年-学期课表
* [X]  获取指定学年-学期成绩
* [X]  计算指定学年-学期gpa

## 待实现功能：

* [ ]  计算所有学期的gpa

## 使用方法：

1. 克隆并在在Pycharm中打开本项目
   ```commandline
   git clone https://github.com/L1nf/gpa.git
   ```
2. 安装相关依赖库
3. （可选）修改 main.py 全局变量中的字段（需要引号），各字段的含义见下表
4. 运行，若未做第4步，亦可在运行的过程中输入各字段（不需要引号）

### 字段含义：

| 字段            | 含义   | 示例                            |
|---------------|------|-------------------------------|
| URL           | 教务网址 | 'http://124.160.107.91:6379/' |
| stu_id        | 学号   | '2090*****'                   |
| pwd           | 密码   | '******'                      |
| academic_year | 学年   | '2022-2023'                   |
| semester      | 学期   | '1'                           |
